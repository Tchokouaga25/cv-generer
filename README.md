# 📄 Générateur de CV Dynamique

Un générateur de CV moderne, interactif et 100% fonctionnel développé en **HTML, CSS et JavaScript** avec **Tailwind CSS**.

---

## 📋 Description du Projet

Ce projet permet de créer rapidement un **CV professionnel** en remplissant un formulaire en temps réel.  
Tout ce que vous saisissez dans le formulaire s’affiche immédiatement dans l’aperçu du CV à droite.

Le CV est **entièrement personnalisable** et peut être **téléchargé en PDF**.

---

## ✨ Fonctionnalités

- **Aperçu en temps réel** : Mise à jour instantanée du CV pendant la saisie
- **Photo de profil** : Upload et prévisualisation de l’image
- **Informations personnelles** :
  - Nom complet, Âge (18-65 ans), Téléphone, Email, Adresse
  - Sexe, Situation matrimoniale
- **Sections dynamiques** avec ajout multiple :
  - Compétences (avec bouton "New Compétence")
  - Formations (Établissement, Diplôme, Année)
  - Expériences professionnelles (Entreprise, Poste, Dates, Missions)
  - Loisirs
- **Langues** : Français & Anglais (niveaux : Moyens, Intermédiaire, Avancé)
- **Validation des champs** :
  - Âge entre 18 et 65 ans
  - Format email valide
  - Numéro de téléphone
- **Gestion du stockage** :
  - Sauvegarder les données dans le **LocalStorage**
  - Restaurer les données
  - Supprimer toutes les données
- **Téléchargement** : Export du CV en **PDF** (via html2pdf.js)
- Design moderne avec **Tailwind CSS** et **Font Awesome**

---

## 🛠️ Technologies Utilisées

- **HTML5**
- **Tailwind CSS** (via CDN)
- **JavaScript (Vanilla)**
- **Font Awesome 6**
- **html2pdf.js** & **html2canvas** (pour l’export PDF)
- **LocalStorage** (persistance des données)

## 🚀 Comment Utiliser

1. **Ouvrir le projet**
   - Double-cliquez sur `index.html` ou ouvrez-le avec Live Server (VS Code)

2. **Remplir le formulaire**
   - Renseignez vos informations personnelles
   - Ajoutez vos compétences, formations, expériences et loisirs
   - La prévisualisation du CV se met à jour automatiquement

3. **Actions disponibles**
   - **Save** → Enregistre les données dans le navigateur
   - **Restore** → Restaure les données sauvegardées
   - **Supprimer** → Efface toutes les données
   - **Download** → Télécharge le CV au format PDF
